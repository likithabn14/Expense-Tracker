Tracker
📌 Overview

Personal Expense Tracker is a web application developed using Flask, Pandas, HTML, CSS, and Bootstrap. It helps users record their daily expenses, organize them by category, and view useful spending summaries through an interactive dashboard.

The application is simple to use and runs completely on the user's computer without requiring any database.

🎯 Objective

The main objective of this project is to help users manage their daily expenses in a simple and organized way. Instead of maintaining expense records manually, users can enter expenses into the application and instantly view summaries such as total spending, highest expense, lowest expense, average expense, top spending category, and total transactions.

❗ Problem Statement

Many people spend money every day without keeping track of their expenses. As a result, it becomes difficult to understand where most of their money is spent. This project provides an easy solution by allowing users to record, edit, search, and manage their expenses in one place.

⚙️ How It Works
Open the application.
Enter the expense details.
Choose a category or create a custom category.
Click Add Expense.
The expense is stored in a CSV file.
The dashboard updates automatically.
Users can edit, delete, search, download, or reset expenses whenever required.
📊 Dashboard Summary

The application provides the following summaries:

Total Expense
Highest Expense
Lowest Expense
Average Expense
Top Spending Category
Total Transactions

Each summary can be viewed by clicking its respective dashboard card.
